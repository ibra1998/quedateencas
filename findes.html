Hasta hace semanas, para algunos de nosotros, era impensable que el gobierno declare el estado de alarma, y que se aplicaran estas medidas en España. Es un cambio drástico temporal en nuestras vidas, y que para algunos, nos supone una gran dificultad. Para no recordar este periodo como tiempo perdido, para no sufrir cada minuto que estés pasando en casa y en fin, para ser felices, debemos dejar de ver esto como un impedimento y empezar a verlo como oportunidad.
Una oportunidad para que se pueda por fin teletrabajar donde sea posible, porque teletrabajar ayuda a conciliar la vida familiar. Para una digitalización de las empresas, para que se mejoren los protocolos de higiene y seguridad alimentaria. Un momento para formarse, para aprender idiomas, para conocernos a nosotros mismos y a nuestras familias.
Este blog, pretende ayudar a cumplir la misión de quedarse en casa.
